doctype -> html의 파일이 어떤 버전인지 정의하는 부분, 현재 표준모드는 html5, 가장 최신버전을 정의하고싶다면

style tag

css를 외부문서에 작성하여 연결하는것이 아니고 html 문서 내부에 작성할 때 사용함 이렇게

script tag

css문서는 link로 불러와야하고 직접작성할때는 style로 작성을 해야했지만 js는 script태그로 js문서를 불러올 수 있고 또한 js코드를 직접 작성도 가능하다.
