css를 어떤 태그를 정하고 그것에 대한 속성과 값을 수정하는것을 말할때 선택자라고 말을 한다.(Selector) css의 특정한 요소를 지칭하는것을 셀렉터 즉 선택자 css의 속성은 properties html의 속성은 attribute라고 칭한다. css의 속성은 div { color : red; } 이런형식으로 스타일을 지정한다. 반드시 ;(세미콜론)을 끝마쳐야함 css의 속성 목록은 이후에 또 설명될것임 댜ㅐ표적으로 color, font-size, width, margin, padding, position 선택자 { 속성(property) : 값; } 이런형식으로 지정을 한다. html의 속성은 attribute라고 칭한다. 선택자의 종류, 속성의 종류의 목록을 학습할 것임 html안에 css태그를 선언하려면 로 선언하면된다. (inline 작성 방식) -> 단점: 일일이 모든 태그에 이렇게 선언을 해주어야한다. 인라인 방식으로 지정을 하는것은 지양해야하는 작성 방식임. 구조적으로 굉장히 불편한 부분, 유지보수할때도 힘든 부분이다. 내장 표현방식(html에 포함하기, embeded 방식) head태그 안에 style형태로 집어넣는거 클래스 선택자 .title { color: red; } .은 클래스 선택자를 의미한다( title이라는 class 선택자를 찾아서 해당 클래스를 갖고 있는 모든 태그의 색상을 변경한다. 속성 종류 # width (가로 너비) -> 단위는 px # height (세로 너비) # font-size (글자 크기) 기본값 16px

width : 300px(웹에서는 300px을 제일 많이 사용함)

실제로는 태그 선택자 div 이렇게 사용하지 않는다. 클래스 선택자로 많이 사용하지 태그 자체로 모든 태그가 적용되기 때문에 태그 선택자는 지양하는 편이다

여백

  • # margin 요소 바깥의 여백
  • margin-top
  • margin-rigth
  • margin-bottom
  • margin-left
  • 이 모든것은 margin의 단축속성, 그리고 개별속성이라고 부른다.
  • 개별속성이 존재하면 단축속성이 있다 -> margin
  • 개별속성 -> top, right, bottom left
  • # padding 안쪽 여백의 값
  • padding 또한 단축속성임
  • top, right, bottom, left margin처럼 사용가능하다.
    • # box2는 box를 상속해서 사용한다.

    색상

    • color : 글자 색상
    • background : 요소의 배경 색상 뿐만 아니라 다른 속성값도 기재가능, 단축속성
    • background-color : 배경 속성의 개별속성, 색상만 지정가능
