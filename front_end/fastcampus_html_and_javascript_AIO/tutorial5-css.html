css를 어떤 태그를 정하고 그것에 대한 속성과 값을 수정하는것을 말할때 선택자라고 말을 한다.(Selector) css의 특정한 요소를 지칭하는것을 셀렉터 즉 선택자 css의 속성은 properties html의 속성은 attribute라고 칭한다. css의 속성은 div { color : red; } 이런형식으로 스타일을 지정한다. 반드시 ;(세미콜론)을 끝마쳐야함 css의 속성 목록은 이후에 또 설명될것임 댜ㅐ표적으로 color, font-size, width, margin, padding, position 선택자 { 속성 : 값; } 이런형식으로 지정을 한다. 선택자의 종류, 속성의 종류의 목록을 학습할 것임 html안에 css태그를 선언하려면 로 선언하면된다. (inline 작성 방식) -> 단점: 일일이 모든 태그에 이렇게 선언을 해주어야한다. 인라인 방식으로 지정을 하는것은 지양해야하는 작성 방식임. 구조적으로 굉장히 불편한 부분, 유지보수할때도 힘든 부분이다. 내장 표현방식(html에 포함하기, embeded 방식) head태그 안에 style형태로 집어넣는거
